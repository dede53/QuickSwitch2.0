QuickSwitch2.0 wiki

QuickSwitch ist eine webbasierte Hausautomatisierungs Software für den Raspberry Pi und alle Server die Node.js unterstützen.

Funktionen

  • PC und Smartphone abhängige Ansicht
  • Schalten von Funksteckdosen vieler Hersteller (u.a.: FRITZ!DECT 200 )
  • Speichern und verwalten von Temperaturen
  • Dimmen von GPIOs des Raspberry Pi
  • Countdowns von Geräten (schalten von Geräten nach einer gewissen Zeit)
  • Zeit und Ereignissteuerung
  • Benutzer abhängige Inhalte
  • Anrufliste aus der Fritzbox
  • Benachrichtigung bei Anrufen über die Fritzbox
  • Chat
  • Synchronisierte Status zwischen den Geräten
  • ...
